# 1단계 설계 문서 — 신호 규칙 · 정답 정의 · 백테스트

> 최종 수정: 2026-07-04 · 상태: **1단계 완료 (D-015)**. 신호 임계값은 가설 초기값으로 확정, 3~4단계에서 데이터 기반 보정

## 0. 전제

- **카테고리**: 뷰티 (✅ 2026-07-04 확정, D-009)
- **페르소나**: 국내 중소 뷰티 브랜드 상품기획/마케팅 담당자
- **분석 단위**: 키워드 × 주(週). 매주 각 키워드에 대해 판정 실행
- **키워드 유형**: 성분 / 제형 / 효능 (브랜드명·제품명은 유니버스에서 제외)

## 1. 신호 규칙

### D — 수요 상승 신호 (네이버 검색어 트렌드)

```
D = (최근 4주 이동평균 ≥ 직전 12주 이동평균 × 1.30)
    AND (최근 4주 평균 ≥ 전년 동일 주차 4주 평균 × 1.15)   # 계절성 필터
```

- 계절성 필터 목적: 매년 반복되는 시즌 수요(예: 5월 선크림)의 오탐 차단
- 임계값 +30% / +15%는 가설 초기값 → 백테스트에서 그리드 탐색으로 보정 예정

### S — 구매 관심 동반 신호 (네이버 쇼핑인사이트)

```
S = (키워드 소속 쇼핑 카테고리의 최근 4주 클릭 추이가 직전 12주 대비 상승)
```

- 논리: 검색만 뜨고 쇼핑이 안 뜨면 "궁금한 것"이지 "사려는 것"이 아님
- 검색↔구매 교차검증 역할
- ⏳ 미결: 키워드→쇼핑 카테고리 매핑 테이블 작성 필요 (2단계에서)

### C — 경쟁 대응 신호 (뉴스 API) — 역방향

```
C = (키워드 ∧ 경쟁 브랜드명 동시 등장 기사 수의 최근 4주 합계
     ≥ 직전 12주 주평균 × 4주 환산 × 2.0)
```

- C가 켜지면 "경쟁사가 이미 움직였다" → 기회의 질 하락
- ✅ 확정(D-011): 경쟁 브랜드는 **키워드별 차등**, 구현은 키워드→세그먼트→브랜드 목록 2단 매핑 (세그먼트 6개 × 브랜드 5~7개). 목록 초안은 02_keyword_universe.md 참조. 브랜드 목록도 백테스트 전 동결 대상

### 판정 로직

```
기회 O  =  D ∧ S ∧ ¬C
```

### 신뢰도 점수 (0~100)

| 구성 | 배점 | 구간 배점 예시 |
|---|---|---|
| D 강도 | 40 | +30%→20점, +45%→30점, +60% 이상→40점 |
| S 강도 | 30 | 상승 확인→15점, +20% 이상→30점 |
| C 부재 | 30 | 완전 부재→30점, 미미한 증가→15점, 급증→0점 |

### 설계 철학: 왜 ML이 아니라 규칙 기반인가 (면접 대비 필수)

1. **설명 가능성**: 판정 근거를 사용자(비개발자 마케터)에게 그대로 노출하는 것이 제품 가치의 일부
2. **과적합 방지**: 키워드 수십 개 × 2년 규모의 데이터에서 ML은 과적합 위험이 크고 검증 신뢰도가 낮음
3. **차별점과의 정합성**: 상용 도구의 블랙박스 AI에 대한 안티테제가 프로젝트 포지셔닝 자체

## 2. 정답(Ground Truth) 정의 — 프로젝트의 심장

```
적중  =  신호 발생 주(t) 이후 4주간 검색량 평균
         ≥ 신호 발생 시점 직전 4주 평균 × 0.90
```

- 해석: 진짜 기회라면 수요가 반짝하고 꺼지지 않고 유지·성장한다
- 반짝 밈(1~2주 폭등 후 소멸)은 브랜드 대응 리드타임이 없으므로 **실패로 분류**

### 명시할 설계 결정

1. **관찰 기간 4주의 근거**: 페르소나가 캠페인/기획으로 대응 가능한 최소 리드타임. 8주 버전을 민감도 분석으로 병기
2. **매출 데이터 부재의 한계**: 기회의 최종 증명은 매출이나 공개 데이터가 없음 → 수요 지속성을 대리 지표(proxy)로 사용함을 명시

## 3. 백테스트 설계

| 항목 | 내용 |
|---|---|
| 키워드 유니버스 | 뷰티 성분/제형/효능 키워드 50~80개, **백테스트 시작 전 확정 후 절대 변경 금지** (선택 편향 차단) |
| 기간 | 최근 2년 (2024.7 ~ 2026.6), 데이터랩 허용 시 3년 |
| 방식 | 매주 롤링로 전 키워드 판정 → 4주 뒤 정답 대조 |
| 주 지표 | **정밀도** (기회 판정 중 실제 적중 비율) — 오탐(헛된 기획 착수) 비용이 크므로 |
| 보조 지표 | 재현율, 신호 발생 빈도(연간 건수) |

### 베이스라인 2종 (비교 없이는 교차검증 가치 증명 불가)

1. **무차별 판정**: 모든 키워드가 항상 기회 → 정밀도 하한선
2. **D 단독 판정**: 검색량 급등 하나만 사용 → "교차검증이 얼마나 개선하는가"의 직접 비교 대상

헤드라인은 "적중률 XX%" 단독이 아니라 "**D 단독 대비 +XX%p**" 형태로 서술한다.
