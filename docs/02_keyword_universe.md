# 키워드 유니버스 (v1.0 — 동결) + 경쟁 브랜드 매핑

> 최종 수정: 2026-07-04 · 상태: **v1.0 동결 (2026-07-04 승인, D-015)**
> 동결 예외 단 1개: 2단계 "검색량 실재 게이트"에서 데이터랩 검색량이 전 기간 사실상 0인 키워드에 한해, 동일 계열 내 교체를 1회 허용 (사전 정의된 규칙이므로 자의적 변경 아님)
> ⚠️ 동결 원칙: 백테스트 시작 전 확정, 이후 추가/삭제 금지 (D-006)

## 0. 유니버스 구성 방법론

### 선정 원칙

1. **체계적 분류 기반**: 뷰티 카테고리의 표준 분류(성분/제형/효능)를 그물망처럼 덮는 방식. 유행 여부와 무관하게 선정
2. **뜬 것 + 안 뜬 것 + 진 것 혼합**: 정체·하락 키워드가 절반 이상 포함되어야 "울리지 말아야 할 곳에서 안 울리는지" 검증 가능
3. **브랜드명·제품명 제외**: 브랜드 검색량은 광고비에 좌우되어 수요 신호로 부적합

### ⚠️ 핵심 개념 구분: 유니버스 키워드 = 소비자 검색어 (표준 성분명 아님)

본 프로젝트의 D 신호는 네이버 검색량을 측정하므로, 유니버스는 소비자가 실제로 입력하는 검색어로 구성한다.
따라서 검증 기준은 "키워드가 표준 성분명과 일치하는가"가 아니라 **"대응하는 표준 성분명/상거래 분류가 실재하는가(매핑 가능한가)"**이다.
예: 소비자 검색어 "시카" ↔ 표준 성분명 "병풀추출물(센텔라아시아티카)". 유니버스에는 "시카"를 유지하고 매핑을 기록한다.

### ⚠️ 선제적으로 명시할 한계: 사후 구성 편향 (Look-ahead Bias)

이 유니버스는 백테스트 기간(2024.7~2026.6)을 겪은 뒤의 지식으로 구성됨. 완화책: 표준 분류 기반 선정, 2024년 이전부터 존재한 일반 키워드 한정, 상승/정체/하락 혼합. 완전 제거는 불가하므로 한계 섹션에 명시.

## 1. 검증 소스 (우선순위 순)

| 순위 | 소스 | 성격 | 용도 |
|---|---|---|---|
| 1 | **대한화장품협회 화장품 성분사전** (kcia.or.kr/cid) | 화장품법 시행규칙이 참조하는 **법적 표준명의 근거 사전**. 성분명·영문명·기원·정의 검색 제공 | 성분 키워드의 표준 성분명 매핑 확인 (1차, 수동) |
| 2 | **식약처 화장품 원료성분정보 Open API** (공공데이터포털) | 표준명·영문명·CAS No·**이명(異名)** 속성 제공, Swagger 명세 공개 | 2단계에서 프로그래매틱 일괄 재검증 (이명 필드로 소비자 검색어↔표준명 매핑 자동화 가능) |
| 3 | **올리브영 카테고리 체계·어워즈** (공개 화면/발표) | 1.8억 건 구매 데이터 기반 40개 부문 시상, 카테고리 분류 공개 | 제형/효능 키워드의 상거래 분류 실재 확인 + 세그먼트 브랜드 근거 |
| 4 | **화해 공개 분류·어워드** | 카테고리/피부고민별 랭킹 분류, 어워드 부문 공개 | 제형/효능 보조 확인 + 브랜드 근거 보조 |

- 소스 발견 사항: 협회 성분사전이 법적 표준의 원천이며 식약처 API는 이를 프로그래매틱하게 쓰는 통로. **식약처 API의 '이명' 필드**가 소비자 검색어→표준명 매핑 자동화에 유용 → 2단계에서 활용
- ⚠️ 역할 제한 유지 (D-013): 위 소스는 존재/분류 검증에만 사용. 트렌드 리포트류를 선정/제외 사유로 사용 금지
